1. 凸集 Convex Sets 期中 Midterm期末 Final

大白话解释 / Plain Language

中文：想象你有一块地，如果这块地上任意两个点之间连一条直线，线上所有的点都还在这块地里面，那这块地就是"凸"的。圆形、三角形是凸的；月牙形不是凸的。

English: A set is convex if for any two points in the set, the entire line segment connecting them also lies within the set. Circles and triangles are convex; crescent shapes are not.

正式定义 / Formal Definition：

集合 $S$ 是凸集（convex set），当且仅当对任意 $x, y \in S$ 和任意 $\theta \in [0,1]$： $$\theta x + (1-\theta)y \in S$$

A set $S$ is convex if and only if for any $x, y \in S$ and any $\theta \in [0,1]$, the convex combination $\theta x + (1-\theta)y \in S$.

$\theta x + (1-\theta)y$ 就是 $x$ 和 $y$ 之间的"混合点"。$\theta$ 从0变到1，这个点就从 $y$ 滑到 $x$。凸集要求这条"滑动路径"上的每个点都在集合里。
The point $\theta x + (1-\theta)y$ "slides" from $y$ to $x$ as $\theta$ goes from 0 to 1. A convex set requires every point on this path to remain in the set.

判断凸集的常用方法 / Methods to Verify Convexity

  1. 定义法 (Definition)：直接验证凸组合封闭性 / Directly verify closure under convex combinations
  2. 凸函数的下水平集 (Sublevel set)：$f$ 凸 → $\{x \mid f(x) \leq c\}$ 是凸集 / If $f$ is convex, then $\{x \mid f(x) \leq c\}$ is convex
  3. 凹函数的上水平集 (Superlevel set)：$f$ 凹 → $\{x \mid f(x) \geq c\}$ 是凸集 / If $f$ is concave, then $\{x \mid f(x) \geq c\}$ is convex

  5. 仿射变换 (Affine transformation)：凸集经仿射映射后仍为凸集

2. 凸函数 Convex Functions 期中期末

大白话解释 / Plain Language

中文：一个函数是凸的，意思就是它的图像像一个碗（开口朝上）。函数图像上任意两点之间连一条线段，这条线段一定在函数图像的上方。

English: A function is convex if its graph looks like a bowl (opening upward). The line segment between any two points on the graph always lies above or on the graph.

零阶条件 / Zeroth-order condition：
$$f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta) f(y), \quad \forall \theta \in [0,1]$$
一阶条件 / First-order condition（需可微 / requires differentiability）：
$$f(y) \geq f(x) + \nabla f(x)^T(y-x), \quad \forall x, y$$

中文：函数图像总是在它的切线（切平面）上方。

The function always lies above its tangent (first-order Taylor approximation).

二阶条件 / Second-order condition（需二阶可微）：
$$\nabla^2 f(x) \succeq 0 \quad (\text{Hessian矩阵半正定 / Hessian is positive semidefinite})$$

中文：函数在每个点的"弯曲方向"都是朝上的。

The function curves upward in every direction at every point.

⚠️ 高频考点 / Key Exam Point：
$f(x,y) = \frac{x^2}{y}$（$y>0$）是凸函数！期中和期末都考了。
$f(x,y) = x^2/y$ (for $y>0$) is convex! Tested in both midterm and final.

Hessian: $H = \frac{2}{y^3}\begin{pmatrix} y^2 & -xy \\ -xy & x^2 \end{pmatrix}$，行列式 $= 0$，对角元素非负 → 半正定 ✓

3. 共轭函数 Conjugate Functions 期中期末

大白话解释 / Plain Language

中文：共轭函数是一种"变换"。对于每个斜率 $y$，找一条斜率为 $y$ 的直线，使得这条直线尽可能地"贴近"原函数的下方，记录最大间隙。

English: The conjugate function is a transform. For each slope $y$, find the maximum gap between the line $yx$ and the function $f(x)$.

定义 / Definition： $$f^*(y) = \sup_x \left[ y^T x - f(x) \right]$$

The conjugate (also called Legendre-Fenchel transform) of $f$ is the supremum of the affine function $y^Tx$ minus $f(x)$.

计算步骤 / Computation Steps

  1. 写出 $g(x) = y^Tx - f(x)$ / Write $g(x) = y^Tx - f(x)$
  2. 对 $x$ 求导令为0 / Take derivative w.r.t. $x$, set to zero
  3. 解出 $x^*$，代回 $g(x)$ / Solve for $x^*$, substitute back
  4. 检查定义域 / Check domain constraints
重要性质 / Key Properties：
  • 共轭函数一定是凸函数（不管原函数凸不凸）/ $f^*$ is always convex
  • 如果 $f$ 是凸的且闭的，则 $f^{**} = f$（双共轭等于自身）/ If $f$ is convex and closed, then $f^{**} = f$
  • Fenchel不等式：$f(x) + f^*(y) \geq x^Ty$ / Fenchel's inequality

4. 超平面定理 Hyperplane Theorems 期中

大白话解释 / Plain Language

中文：超平面是高维空间里的"平面"（二维里是直线，三维里是平面）。支撑超平面像用平板"贴"凸体边界；分离超平面像用一刀切开两块不重叠的橡皮泥。

English: A hyperplane is a flat surface in high dimensions. A supporting hyperplane "touches" the boundary of a convex set. A separating hyperplane "cuts" between two non-overlapping convex sets.

支撑超平面定理 / Supporting Hyperplane Theorem：
如果 $x_0$ 是凸集 $C$ 的边界点，则存在 $a \neq 0$ 使得 $a^Tx \leq a^Tx_0, \forall x \in C$。
If $x_0$ is a boundary point of convex set $C$, there exists $a \neq 0$ such that $a^Tx \leq a^Tx_0$ for all $x \in C$.
分离超平面定理 / Separating Hyperplane Theorem：
如果 $C, D$ 是不相交的凸集，则存在 $a \neq 0, b$ 使得： $$a^Tx \leq b, \forall x \in C; \quad a^Tx \geq b, \forall x \in D$$ If $C$ and $D$ are disjoint convex sets, there exists a hyperplane $a^Tx = b$ separating them.
凸集 = 半空间的交集 / Convex set = Intersection of half-spaces：
任何闭凸集都可以写成（可能无穷多个）半空间的交集。
Any closed convex set is the intersection of (possibly infinitely many) half-spaces.

5. 择一定理 Theorems of Alternatives PPT补充

大白话解释 / Plain Language

中文：择一定理说的是：两个线性系统，恰好有一个有解。就像"要么门开着，要么门关着"——两种情况必居其一，不可能同时成立。

English: Theorems of alternatives state that exactly one of two systems of inequalities has a solution — they are mutually exclusive and exhaustive.

Farkas引理 / Farkas' Lemma：
恰好有以下之一成立：
(I) 存在 $x$ 使得 $Ax \leq b$
(II) 存在 $y \geq 0$ 使得 $A^Ty = 0, b^Ty < 0$

Exactly one of the following holds:
(I) There exists $x$ such that $Ax \leq b$
(II) There exists $y \geq 0$ such that $A^Ty = 0$ and $b^Ty < 0$
Farkas引理的直觉：如果线性不等式系统 $Ax \leq b$ 无解，那么一定能找到一个"证据"（$y$）来证明它无解——通过对不等式做非负线性组合得到矛盾 $0 \leq$ 负数。
If the system $Ax \leq b$ has no solution, there must exist a "certificate of infeasibility" — a non-negative combination of the inequalities that yields a contradiction.
应用 / Applications：
  • 证明强对偶性 / Proving strong duality
  • 推导KKT条件的必要性 / Deriving necessity of KKT conditions
  • 证明分离超平面定理 / Proving separating hyperplane theorem

6. 拟凸函数 Quasiconvex Functions PPT补充

大白话解释 / Plain Language

中文：拟凸函数比凸函数"弱"一些。它不要求函数像碗，只要求所有下水平集都是凸的。换句话说，函数值低于某个阈值的区域总是"一整块"的，不会分成两块。

English: A quasiconvex function is weaker than convex — it only requires all sublevel sets to be convex. The region where $f(x) \leq c$ is always "one piece" (connected and convex).

定义 / Definition：
$f$ 是拟凸的（quasiconvex），如果所有下水平集 $S_\alpha = \{x \mid f(x) \leq \alpha\}$ 都是凸集。
等价条件：$f(\theta x + (1-\theta)y) \leq \max\{f(x), f(y)\}$

$f$ is quasiconvex if all sublevel sets are convex, equivalently: $f(\theta x + (1-\theta)y) \leq \max\{f(x), f(y)\}$
凸 vs 拟凸 / Convex vs Quasiconvex：
  • 凸函数一定是拟凸的 / Every convex function is quasiconvex
  • 拟凸不一定是凸的 / Quasiconvex does not imply convex
  • 例：$f(x) = \lceil x \rceil$（取整函数）是拟凸但不是凸的
  • $\sqrt{x}$（$x \geq 0$）是拟凸且拟凹的（即拟线性 quasilinear）

7. 凸组合与极值性质 Convex Combination & Extrema 期中

大白话解释 / Plain Language

中文：凸组合就是"加权平均"（权重非负且和为1）。凸包就是用橡皮筋把一堆点围起来的区域。

English: A convex combination is a weighted average with non-negative weights summing to 1. The convex hull is like stretching a rubber band around a set of points.

核心定理 / Key Theorem：凸函数在凸包上的最大值在顶点取到。
$$\max_{x \in \text{conv}(v_1,\ldots,v_n)} f(x) = \max_{1 \leq i \leq n} f(v_i)$$ A convex function attains its maximum over a convex hull at one of the vertices.
碗形函数在多边形区域上的最高点，一定在多边形的某个角（顶点）上。内部点是顶点的加权平均，由凸性，加权平均的函数值 ≤ 函数值的加权平均 ≤ 最大函数值。
A bowl-shaped function reaches its peak on a polygon at a corner. Interior points are averages of vertices, and by convexity, $f(\text{average}) \leq \text{average of } f \leq \max f$.

8. 标准凸优化问题 Standard Convex Problem PPT补充

大白话解释 / Plain Language

中文：凸优化问题就是在凸集上最小化凸函数。它的好处是：局部最优 = 全局最优，不会陷入"假的谷底"。

English: A convex optimization problem minimizes a convex function over a convex set. The key benefit: any local minimum is a global minimum.

标准形式 / Standard Form： $$\begin{aligned} \text{minimize} \quad & f_0(x) \\ \text{subject to} \quad & f_i(x) \leq 0, \quad i = 1,\ldots,m \\ & Ax = b \end{aligned}$$ 其中 $f_0, f_1, \ldots, f_m$ 都是凸函数。
where $f_0, f_1, \ldots, f_m$ are all convex functions.

常见凸优化问题类型 / Common Types

🎨 图示：凸优化问题层级关系 / Convex Optimization Problem Hierarchy

  • 线性规划 LP (Linear Programming)：目标和约束都是线性的
  • 二次规划 QP (Quadratic Programming)：二次目标 + 线性约束
  • 二阶锥规划 SOCP (Second-Order Cone Programming)：含范数约束
  • 半定规划 SDP (Semidefinite Programming)：矩阵变量 + 半正定约束

问题转化技巧 / Problem Transformation

  • 引入松弛变量 / Introduce slack variables
  • 等价变换（如取对数）/ Equivalent transformations (e.g., taking log)
  • 引入辅助变量使约束线性化 / Introduce auxiliary variables to linearize
  • Epigraph形式：minimize $t$ s.t. $f(x) \leq t$ / Epigraph reformulation

9. 对偶理论 Duality Theory 期末 Final

大白话解释 / Plain Language

中文：你有一个优化问题（原问题），对偶理论帮你构造一个"镜像问题"（对偶问题）。对偶问题给出原问题最优值的一个界，很多情况下这个界是紧的。

English: Duality constructs a "mirror problem" (dual) from the original (primal). The dual provides a bound on the primal optimal value, and often this bound is tight.

拉格朗日函数 / Lagrangian

对于问题 minimize $f_0(x)$ s.t. $f_i(x) \leq 0$, $h_j(x) = 0$： $$L(x, \lambda, \nu) = f_0(x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{j=1}^p \nu_j h_j(x)$$ 其中 $\lambda_i \geq 0$ 是不等式约束的乘子，$\nu_j$ 是等式约束的乘子。
$\lambda_i \geq 0$ are multipliers for inequality constraints; $\nu_j$ are multipliers for equality constraints.

对偶函数与对偶问题 / Dual Function & Dual Problem

对偶函数 / Dual function： $$g(\lambda, \nu) = \inf_x L(x, \lambda, \nu)$$ 对偶问题 / Dual problem： maximize $g(\lambda, \nu)$ s.t. $\lambda \geq 0$

The dual function is always concave (even if primal is non-convex). The dual problem maximizes this lower bound.

KKT条件 / KKT Conditions

中文：KKT条件是"最优解必须满足的条件清单"：
  1. 原始可行性 Primal feasibility：$f_i(x^*) \leq 0$, $h_j(x^*) = 0$
  2. 对偶可行性 Dual feasibility：$\lambda_i^* \geq 0$
  3. 互补松弛 Complementary slackness：$\lambda_i^* f_i(x^*) = 0$
  4. 梯度条件 Stationarity：$\nabla f_0(x^*) + \sum \lambda_i^* \nabla f_i(x^*) + \sum \nu_j^* \nabla h_j(x^*) = 0$

KKT conditions are necessary for optimality (and sufficient for convex problems).

📋 KKT条件检查清单 / KKT Conditions Checklist

弱对偶与强对偶 / Weak & Strong Duality

弱对偶 Weak duality：$d^* \leq p^*$（总是成立 / always holds）
强对偶 Strong duality：$d^* = p^*$（需要条件 / requires conditions）

Slater条件 / Slater's condition：如果原问题是凸的，且存在严格可行点（$f_i(x) < 0$ 对所有 $i$），则强对偶成立。
If the primal is convex and there exists a strictly feasible point, strong duality holds.

10. 灵敏度分析 Sensitivity Analysis PPT补充

大白话解释 / Plain Language

中文：灵敏度分析回答的问题是：如果我稍微放松一下约束条件，最优值会变化多少？答案就藏在最优对偶变量（拉格朗日乘子）里。

English: Sensitivity analysis answers: how much does the optimal value change if we slightly relax a constraint? The answer is encoded in the optimal dual variables (Lagrange multipliers).

核心结论 / Key Result：
设 $p^*(u, v)$ 是扰动问题（约束变为 $f_i(x) \leq u_i$, $h_j(x) = v_j$）的最优值，则在强对偶成立时： $$\lambda_i^* = -\frac{\partial p^*}{\partial u_i}, \quad \nu_j^* = -\frac{\partial p^*}{\partial v_j}$$ The optimal multiplier $\lambda_i^*$ gives the rate of change of the optimal value with respect to tightening/relaxing the $i$-th constraint.
拉格朗日乘子就是约束的"影子价格"。$\lambda_i^*$ 越大，说明第 $i$ 个约束越"紧"，放松它能带来越大的收益。
The multiplier is the "shadow price" of a constraint. A larger $\lambda_i^*$ means relaxing constraint $i$ yields more improvement.

11. 梯度下降法 Gradient Descent 期末

大白话解释 / Plain Language

中文：梯度下降就是"下山"策略：站在山上，看哪个方向最陡，朝反方向走一步。重复直到到达谷底。

English: Gradient descent is a "go downhill" strategy: at each point, find the steepest direction and take a step in the opposite direction. Repeat until reaching the bottom.

算法 / Algorithm：
  1. 给定初始点 $x^{(0)}$ / Given starting point $x^{(0)}$
  2. 重复 / Repeat：
    • $\Delta x = -\nabla f(x)$ （下降方向 / descent direction）
    • 线搜索确定步长 $t$ / Line search for step size $t$
    • $x := x + t\Delta x$ （更新 / update）
  3. 直到 $\|\nabla f(x)\|_2 \leq \epsilon$ / Until $\|\nabla f(x)\|_2 \leq \epsilon$

线搜索 / Line Search

精确线搜索 / Exact line search： $$t = \arg\min_{t > 0} f(x + t\Delta x)$$ Choose the step size that minimizes $f$ along the search direction.
回溯线搜索 / Backtracking line search：
参数 / Parameters: $\alpha \in (0, 0.5)$, $\beta \in (0, 1)$
从 $t=1$ 开始，重复 $t := \beta t$ 直到 / Start with $t=1$, repeat $t := \beta t$ until: $$f(x + t\Delta x) \leq f(x) + \alpha t \nabla f(x)^T \Delta x$$

This is called the Armijo condition. It ensures "sufficient decrease".

回溯线搜索就像下山时先迈大步，如果发现没怎么下降就缩短步幅重试。$\alpha$ 控制"什么叫足够好的下降"，$\beta$ 控制"每次缩短多少"。
Start with a big step; if the decrease isn't enough, shrink the step. $\alpha$ controls "what counts as enough decrease"; $\beta$ controls "how much to shrink".

12. 最速下降方向 Steepest Descent Direction 期末

大白话解释 / Plain Language

中文："在当前点，哪个方向走一小步能让函数值下降最快？"答案取决于你怎么度量"一步"的大小。

English: "Which direction gives the fastest decrease for a unit step?" The answer depends on how you measure "unit step" (which norm you use).

归一化最速下降方向 / Normalized steepest descent direction： $$\Delta x_{\text{nsd}} = \arg\min_{\|v\| = 1} \nabla f(x)^T v$$ 非归一化 / Unnormalized： $$\Delta x_{\text{sd}} = \|\nabla f(x)\|_* \cdot \Delta x_{\text{nsd}}$$ 其中 $\|\cdot\|_*$ 是对偶范数 / where $\|\cdot\|_*$ is the dual norm.
不同范数下的结果 / Results for different norms：
  • $\ell_2$ 范数：$\Delta x_{\text{nsd}} = -\nabla f(x)/\|\nabla f(x)\|_2$（普通梯度下降 / standard gradient descent）
  • $\ell_1$ 范数：$\Delta x_{\text{nsd}} = -\text{sign}(\partial f/\partial x_i) e_i$，选梯度绝对值最大的坐标方向 / coordinate with largest $|\partial f/\partial x_i|$
  • $P$-范数 $\|v\|_P = (v^TPv)^{1/2}$：$\Delta x_{\text{nsd}} = -P^{-1}\nabla f(x) / (\nabla f(x)^T P^{-1} \nabla f(x))^{1/2}$

13. 牛顿法 Newton's Method 期末

大白话解释 / Plain Language

中文：梯度下降只看"坡度"（一阶），牛顿法还看"地形弯曲"（二阶）。它在当前点用二次函数近似原函数，直接跳到二次函数的最低点。

English: Gradient descent uses only first-order info; Newton's method also uses second-order (curvature). It approximates $f$ with a quadratic and jumps to its minimum.

牛顿下降方向 / Newton step： $$\Delta x_{\text{nt}} = -[\nabla^2 f(x)]^{-1} \nabla f(x)$$ The Newton step is the negative Hessian-inverse times the gradient.
牛顿减量 / Newton decrement： $$\lambda(x) = \left(\nabla f(x)^T [\nabla^2 f(x)]^{-1} \nabla f(x)\right)^{1/2} = \left(\Delta x_{\text{nt}}^T \nabla^2 f(x) \Delta x_{\text{nt}}\right)^{1/2}$$

中文：衡量"当前点离最优点还有多远"的仿射不变量。

An affine-invariant measure of the distance to optimality.

14. 自协调函数 Self-concordant Functions 期末

English: Self-concordant functions have their third derivative controlled by the second derivative. This guarantees Newton's method converges without needing problem-specific parameters (like Lipschitz constants).

定义 / Definition（一维 / univariate）： $$|f'''(x)| \leq 2\, f''(x)^{3/2}$$ 多维 / Multivariate：对所有方向 $v$： $$|D^3 f(x)[v,v,v]| \leq 2\,(v^T \nabla^2 f(x)\, v)^{3/2}$$ The third directional derivative is bounded by the 3/2 power of the second directional derivative.
经典例子 / Classic example：$f(x) = -\ln(x)$
$f''(x) = 1/x^2$, $f'''(x) = -2/x^3$
$|f'''| = 2/x^3 = 2(1/x^2)^{3/2} = 2(f'')^{3/2}$ ✓
The negative logarithm is self-concordant (with equality in the bound).
为什么重要 / Why it matters：

  • 对数障碍函数是自协调的 → 内点法的理论基础 / Log barrier is self-concordant → foundation of interior-point methods

English: Ordinary convex functions can be "flat" at the bottom. Strong convexity guarantees a minimum curvature (at least $m/2 \cdot \|x\|^2$), ensuring a unique minimizer and faster convergence.

16. 等式约束牛顿法 Equality Constrained Newton PPT补充

大白话解释 / Plain Language

中文：当优化问题有等式约束 $Ax=b$ 时，不能随便走牛顿步（可能违反约束）。需要在满足约束的"平面"上做牛顿法。

English: When there are equality constraints $Ax=b$, we can't take arbitrary Newton steps. We must stay on the feasible "plane" while applying Newton's method.

KKT系统 / KKT System：等式约束牛顿步通过求解以下系统得到： $$\begin{pmatrix} \nabla^2 f(x) & A^T \\ A & 0 \end{pmatrix} \begin{pmatrix} \Delta x_{\text{nt}} \\ w \end{pmatrix} = \begin{pmatrix} -\nabla f(x) \\ 0 \end{pmatrix}$$ The Newton step for equality-constrained problems is found by solving this KKT system.

不可行牛顿法 / Infeasible Start Newton Method

中文：如果初始点不满足 $Ax=b$，可以用不可行牛顿法——同时减小目标函数和约束违反量。

If the starting point doesn't satisfy $Ax=b$, the infeasible start Newton method simultaneously reduces the objective and the constraint violation.


KKT系统变为： $$\begin{pmatrix} \nabla^2 f(x) & A^T \\ A & 0 \end{pmatrix} \begin{pmatrix} \Delta x \\ w \end{pmatrix} = \begin{pmatrix} -\nabla f(x) \\ Ax - b \end{pmatrix}$$ The right-hand side now includes the residual $Ax-b$ to drive feasibility.

17. 障碍方法 Barrier Method (Interior Point) PPT补充

大白话解释 / Plain Language

中文：障碍方法处理不等式约束的思路是：在可行域边界设置"电网"（障碍函数），让优化过程自动远离边界。然后逐渐降低"电压"，让解慢慢靠近真正的最优点。

English: The barrier method handles inequality constraints by placing an "electric fence" (barrier function) at the boundary. The optimizer stays away from the boundary, and we gradually reduce the "voltage" to approach the true optimum.

对数障碍函数 / Logarithmic Barrier： $$\phi(x) = -\sum_{i=1}^m \ln(-f_i(x))$$ 当 $x$ 接近约束边界（$f_i(x) \to 0$）时，$\phi(x) \to +\infty$。
As $x$ approaches the constraint boundary, the barrier goes to infinity, preventing the optimizer from leaving the feasible region.
中心路径 / Central Path：
对每个 $t > 0$，定义中心点 $x^*(t)$ 为以下问题的解： $$\text{minimize} \quad tf_0(x) + \phi(x)$$ 当 $t \to \infty$ 时，$x^*(t) \to x^*$（原问题最优解）。
For each $t > 0$, the centering point $x^*(t)$ minimizes $tf_0(x) + \phi(x)$. As $t \to \infty$, $x^*(t)$ converges to the primal optimum.
障碍方法算法 / Barrier Method Algorithm：
  1. 给定初始可行点 $x$，$t := t^{(0)} > 0$，增长因子 $\mu > 1$ / Given feasible $x$, $t := t^{(0)}$, $\mu > 1$
  2. 重复 / Repeat：
    • 中心化步 Centering：用牛顿法求解 minimize $tf_0(x) + \phi(x)$
    • 更新 Update：$t := \mu t$（增大 $t$）
  3. 直到 $m/t \leq \epsilon$（对偶间隙足够小）/ Until $m/t \leq \epsilon$
对偶间隙 / Duality gap：$f_0(x^*(t)) - p^* \leq m/t$
The suboptimality of the centering point is bounded by $m/t$, where $m$ is the number of inequality constraints.

Problem 1: 判断凸集 / Determine if $S$ is convex

题目 / Problem：$S = \{x \in \mathbb{R}^n \mid x_1 x_2 \cdots x_n \geq 1, x_i > 0\}$，判断是否为凸集。
解答 / Solution：

结论：$S$ 是凸集。/ $S$ is convex.

中文证明：
取对数变换：条件 $x_1 x_2 \cdots x_n \geq 1$ 等价于 $\ln x_1 + \ln x_2 + \cdots + \ln x_n \geq 0$。
定义 $g(x) = \ln x_1 + \ln x_2 + \cdots + \ln x_n$。由于 $\ln$ 是凹函数，$g(x)$ 是凹函数的和，也是凹函数。
$S = \{x \in \mathbb{R}^n_{++} \mid g(x) \geq 0\}$ 是凹函数的上水平集（superlevel set），因此是凸集。✓

English Proof：
Taking logarithm: $x_1 \cdots x_n \geq 1 \Leftrightarrow \sum_{i=1}^n \ln x_i \geq 0$.
Let $g(x) = \sum \ln x_i$. Since $\ln$ is concave, $g$ is concave (sum of concave functions).
$S = \{x \in \mathbb{R}^n_{++} \mid g(x) \geq 0\}$ is a superlevel set of a concave function, hence convex. ✓

Problem 2(a): 凸函数定义 / Definition of convex function

中文：函数 $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，如果 $\text{dom}(f)$ 是凸集，且对任意 $x, y \in \text{dom}(f)$，任意 $\theta \in [0,1]$： $$f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)$$ English：A function $f: \mathbb{R}^n \to \mathbb{R}$ is convex if $\text{dom}(f)$ is a convex set and for all $x, y \in \text{dom}(f)$, for all $\theta \in [0,1]$: $$f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)$$

Problem 2(b): 共轭函数定义 / Definition of conjugate function

中文：函数 $f$ 的共轭函数 $f^*: \mathbb{R}^n \to \mathbb{R}$ 定义为： $$f^*(y) = \sup_{x \in \text{dom}(f)} \left(y^T x - f(x)\right)$$ English：The conjugate of $f$ is $f^*(y) = \sup_{x \in \text{dom}(f)} (y^Tx - f(x))$.

Problem 2(c): 支撑超平面定理 / Supporting hyperplane theorem

中文：设 $C$ 是非空凸集，$x_0$ 是 $C$ 的边界点。则存在非零向量 $a$ 使得对所有 $x \in C$，$a^T x \leq a^T x_0$。即超平面 $\{x \mid a^Tx = a^Tx_0\}$ 是 $C$ 在 $x_0$ 处的支撑超平面。

English：Let $C$ be a nonempty convex set and $x_0$ a boundary point of $C$. Then there exists $a \neq 0$ such that $a^Tx \leq a^Tx_0$ for all $x \in C$. The hyperplane $\{x \mid a^Tx = a^Tx_0\}$ is a supporting hyperplane to $C$ at $x_0$.

Problem 2(d): 圆盘写成半空间交集 / Disk as intersection of half-spaces

题目：$S = \{(x_1,x_2) \mid x_1^2 + x_2^2 \leq 1\}$

中文解答：
对圆盘边界上每个点 $a = (a_1, a_2)$（满足 $a_1^2 + a_2^2 = 1$），对应的支撑超平面为 $a_1 x_1 + a_2 x_2 = 1$，半空间为 $a_1 x_1 + a_2 x_2 \leq 1$。
圆盘是所有这些半空间的交集： $$S = \bigcap_{\|a\|_2 = 1} \{x \mid a^T x \leq 1\}$$ English：
For each boundary point $a = (a_1, a_2)$ with $\|a\|=1$, the supporting hyperplane is $a^Tx = 1$ and the half-space is $a^Tx \leq 1$. The disk is: $$S = \bigcap_{\|a\|_2 = 1} \{x \mid a^T x \leq 1\}$$

Problem 2(e): 找分离超平面 / Find separating hyperplane

题目：$S_1 = \{(x_1,x_2) \mid (x_1-1)^2 + x_2^2 \leq 1\}$, $S_2 = \{(x_1,x_2) \mid x_2 \geq 2\}$

中文解答：
$S_1$ 是圆心 $(1,0)$、半径 $1$ 的圆盘，其最高点为 $(1,1)$（$x_2$ 最大值为 $1$）。
$S_2$ 是 $x_2 \geq 2$ 的半平面，最低处为 $x_2 = 2$。
因此 $S_1$ 中所有点满足 $x_2 \leq 1$，$S_2$ 中所有点满足 $x_2 \geq 2$。
分离超平面：$x_2 = c$，其中 $1 \leq c \leq 2$。例如 $\boxed{x_2 = 1.5}$。

English：
$S_1$ is a disk centered at $(1,0)$ with radius $1$; its highest point is $(1,1)$.
$S_2$ is the half-plane $x_2 \geq 2$.
All points in $S_1$ have $x_2 \leq 1$; all points in $S_2$ have $x_2 \geq 2$.
Separating hyperplane: $x_2 = c$ for any $c \in [1, 2]$, e.g., $\boxed{x_2 = 1.5}$.

Problem 3(a): $f(x) = 3x - 4$ 的共轭 / Conjugate of $f(x) = 3x-4$

中文：
$f^*(y) = \sup_x [yx - (3x-4)] = \sup_x [(y-3)x + 4]$
这是关于 $x$ 的线性函数。当 $y \neq 3$ 时，上确界为 $+\infty$；当 $y = 3$ 时，上确界为 $4$。
$$\boxed{f^*(y) = \begin{cases} 4 & \text{if } y = 3 \\ +\infty & \text{if } y \neq 3 \end{cases}}$$ English：
$f^*(y) = \sup_x [yx - 3x + 4] = \sup_x [(y-3)x + 4]$.
This is linear in $x$. If $y \neq 3$, the supremum is $+\infty$. If $y = 3$, the supremum is $4$.
$$\boxed{f^*(y) = \begin{cases} 4 & y = 3 \\ +\infty & y \neq 3 \end{cases}}$$

Problem 3(b): $f(x) = e^x$ 的共轭 / Conjugate of $f(x) = e^x$

中文：
$f^*(y) = \sup_x [yx - e^x]$
令 $\frac{d}{dx}[yx - e^x] = y - e^x = 0$，得 $x = \ln y$（需要 $y > 0$）。
代入：$f^*(y) = y\ln y - y$
当 $y = 0$ 时：$\sup_x [-e^x] = 0$（当 $x \to -\infty$）
当 $y < 0$ 时：$yx - e^x \to -\infty$ 对所有 $x$... 实际上 $\sup_x[yx - e^x]$ 当 $y<0$ 时，令 $x \to -\infty$ 得 $0$，但 $yx \to +\infty$... 不对，$x \to -\infty$ 时 $yx \to +\infty$（$y<0$），$e^x \to 0$，所以上确界为 $+\infty$？
修正：当 $y < 0$，取 $x \to -\infty$：$yx \to +\infty$，$e^x \to 0$，所以 $yx - e^x \to +\infty$。
因此 $\text{dom}(f^*) = \{y \geq 0\}$... 再检查：$y < 0$, $x \to -\infty$: $yx = y \cdot(-\infty) = +\infty$（因为 $y<0$），所以确实 $f^*(y) = +\infty$。

$$\boxed{f^*(y) = \begin{cases} y\ln y - y & y > 0 \\ 0 & y = 0 \\ +\infty & y < 0 \end{cases}}$$ English：
$f^*(y) = \sup_x[yx - e^x]$. Setting derivative to zero: $y - e^x = 0 \Rightarrow x = \ln y$ (requires $y > 0$).
Substituting: $f^*(y) = y\ln y - y$ for $y > 0$.
For $y = 0$: $\sup_x[-e^x] = 0$ (as $x \to -\infty$).
For $y < 0$: as $x \to -\infty$, $yx \to +\infty$ and $e^x \to 0$, so $f^*(y) = +\infty$.
$$\boxed{f^*(y) = \begin{cases} y\ln y - y & y > 0 \\ 0 & y = 0 \end{cases} \quad \text{dom}(f^*) = \mathbb{R}_+}$$

Problem 4: 凸函数在凸包上最大值 / Max of convex function on convex hull

题目：证明 $\max_{x \in S} f(x) = f(v_k)$，其中 $S$ 是 $\{v_1,\ldots,v_n\}$ 的凸包，$f(v_k) = \max_i f(v_i)$。

中文证明：
对任意 $x \in S$，存在 $\lambda_i \geq 0$，$\sum \lambda_i = 1$ 使得 $x = \sum_{i=1}^n \lambda_i v_i$。
由 $f$ 的凸性（Jensen不等式）： $$f(x) = f\left(\sum_{i=1}^n \lambda_i v_i\right) \leq \sum_{i=1}^n \lambda_i f(v_i) \leq \sum_{i=1}^n \lambda_i \cdot f(v_k) = f(v_k)$$ 因此 $\max_{x \in S} f(x) \leq f(v_k)$。又因为 $v_k \in S$，所以 $\max_{x \in S} f(x) \geq f(v_k)$。
综合得 $\max_{x \in S} f(x) = f(v_k)$。✓

English Proof：
For any $x \in S$, there exist $\lambda_i \geq 0$ with $\sum \lambda_i = 1$ such that $x = \sum \lambda_i v_i$.
By convexity of $f$ (Jensen's inequality): $$f(x) = f\left(\sum \lambda_i v_i\right) \leq \sum \lambda_i f(v_i) \leq \sum \lambda_i f(v_k) = f(v_k)$$ Thus $\max_{x \in S} f(x) \leq f(v_k)$. Since $v_k \in S$, we also have $\max_{x \in S} f(x) \geq f(v_k)$.
Therefore $\max_{x \in S} f(x) = f(v_k)$. ✓

Problem 5(a): 证明 $f(y_1,y_2) = y_1^2/y_2$ 是凸函数

题目：$f(y_1, y_2) = \frac{y_1^2}{y_2}$，$(y_1, y_2) \in \mathbb{R} \times \mathbb{R}_{++}$，证明凸性。

中文证明（Hessian法）：
计算偏导数： $$\frac{\partial f}{\partial y_1} = \frac{2y_1}{y_2}, \quad \frac{\partial f}{\partial y_2} = -\frac{y_1^2}{y_2^2}$$ $$\frac{\partial^2 f}{\partial y_1^2} = \frac{2}{y_2}, \quad \frac{\partial^2 f}{\partial y_1 \partial y_2} = -\frac{2y_1}{y_2^2}, \quad \frac{\partial^2 f}{\partial y_2^2} = \frac{2y_1^2}{y_2^3}$$ Hessian矩阵： $$\nabla^2 f = \frac{2}{y_2^3}\begin{pmatrix} y_2^2 & -y_1 y_2 \\ -y_1 y_2 & y_1^2 \end{pmatrix}$$ 验证半正定：$\frac{2}{y_2^3} > 0$（因为 $y_2 > 0$），矩阵 $\begin{pmatrix} y_2^2 & -y_1 y_2 \\ -y_1 y_2 & y_1^2 \end{pmatrix}$ 的行列式 $= y_2^2 \cdot y_1^2 - y_1^2 y_2^2 = 0 \geq 0$，对角元素非负。所以 Hessian 半正定，$f$ 是凸函数。✓

English Proof (Hessian method)：
The Hessian is $\nabla^2 f = \frac{2}{y_2^3}\begin{pmatrix} y_2^2 & -y_1 y_2 \\ -y_1 y_2 & y_1^2 \end{pmatrix}$.
Since $y_2 > 0$, the scalar $2/y_2^3 > 0$. The matrix has determinant $= 0 \geq 0$ and non-negative diagonal entries, so it is PSD. Hence $f$ is convex. ✓

另解（透视函数法）/ Alternative (Perspective)：
$g(y_1) = y_1^2$ 是凸函数。$f(y_1, y_2) = y_2 \cdot g(y_1/y_2) = y_2 \cdot (y_1/y_2)^2 = y_1^2/y_2$ 是 $g$ 的透视函数，因此凸。✓
$f(y_1,y_2) = y_2 \cdot g(y_1/y_2)$ is the perspective of $g(y_1) = y_1^2$, which preserves convexity. ✓

Problem 5(b): 证明 $h(x) = f(x)^2/g(x)$ 是凸函数

题目：$f(x)$ 正凸，$g(x)$ 正凹，证明 $h(x) = f(x)^2/g(x)$ 凸。

中文证明：

因此 $h(x)$ 是凸函数。✓

Therefore $h(x)$ is convex. ✓

(b) $f(x) = \frac{1}{x_1 x_2}$, $x \in \mathbb{R}^2_{++}$

结论：凸函数（Convex）。

中文：$f(x) = (x_1 x_2)^{-1}$。可以验证 Hessian 半正定。
$\nabla^2 f = \begin{pmatrix} 2/(x_1^3 x_2) & 1/(x_1^2 x_2^2) \\ 1/(x_1^2 x_2^2) & 2/(x_1 x_2^3) \end{pmatrix}$
行列式 $= 4/(x_1^4 x_2^4) - 1/(x_1^4 x_2^4) = 3/(x_1^4 x_2^4) > 0$，对角元素正 → 正定 → 凸。

English：The Hessian has positive diagonal entries and positive determinant $3/(x_1^4 x_2^4) > 0$, so it is positive definite. Hence $f$ is (strictly) convex.

(c) $f(x,y) = x/y$, $x, y \in \mathbb{R}_+$

结论：既不凸也不凹（Neither convex nor concave）。

中文：Hessian为 $\nabla^2 f = \begin{pmatrix} 0 & -1/y^2 \\ -1/y^2 & 2x/y^3 \end{pmatrix}$。
行列式 $= 0 - 1/y^4 = -1/y^4 < 0$ → Hessian不定 → 既不凸也不凹。

English：The Hessian has determinant $-1/y^4 < 0$, so it is indefinite. $f$ is neither convex nor concave.

(d) $f(x,y) = x^2/y$, $x, y \in \mathbb{R}_+$ (with $y > 0$)

结论：凸函数（Convex）。

中文/English：见期中 Problem 5(a) 的证明。这是 $g(x)=x^2$ 的透视函数，凸性保持。
See Midterm Problem 5(a). This is the perspective of $g(x)=x^2$, which preserves convexity.

(e) $f(x) = \|Ax - b\|$, $A \in \mathbb{R}^{m \times n}$, $b \in \mathbb{R}^m$

结论：凸函数（Convex）。

中文：范数 $\|\cdot\|$ 是凸函数，$Ax-b$ 是 $x$ 的仿射函数。凸函数的仿射复合仍为凸函数。

English：Any norm $\|\cdot\|$ is convex, and $Ax-b$ is affine in $x$. Composition of a convex function with an affine mapping preserves convexity.

Problem 2: 计算共轭函数 / Compute conjugate functions

(a) $f(x) = x_1^2 + x_2^2$, $x \in \mathbb{R}^2_{++}$

中文：
$f^*(y) = \sup_x [y^Tx - (x_1^2 + x_2^2)] = \sup_x [y_1 x_1 + y_2 x_2 - x_1^2 - x_2^2]$
对 $x_1$ 求导：$y_1 - 2x_1 = 0 \Rightarrow x_1 = y_1/2$
对 $x_2$ 求导：$y_2 - 2x_2 = 0 \Rightarrow x_2 = y_2/2$
注意定义域约束 $x \in \mathbb{R}^2_{++}$，需要 $y_1 > 0, y_2 > 0$（否则最优 $x$ 在边界，需要单独分析）。
当 $y_1, y_2 > 0$ 时：代入得 $f^*(y) = \frac{y_1^2}{4} + \frac{y_2^2}{4} = \frac{\|y\|^2}{4}$
当 $y_1 \leq 0$ 或 $y_2 \leq 0$ 时：取 $x_i \to 0^+$，$f^*(y) = \frac{\|y\|^2}{4}$（因为无约束最优点不在定义域内时，需要在边界取值）。
实际上，由于 $\mathbb{R}^2_{++}$ 是开集，当 $y_i \leq 0$ 时上确界仍可通过取极限得到。

$$\boxed{f^*(y) = \frac{y_1^2 + y_2^2}{4} = \frac{\|y\|_2^2}{4}, \quad y \in \mathbb{R}^2}$$ （注：如果严格限制 $x \in \mathbb{R}^2_{++}$，当 $y_i \leq 0$ 时上确界在边界趋近取到）

English：
$f^*(y) = \sup_{x \in \mathbb{R}^2_{++}}[y^Tx - \|x\|^2]$. Setting gradient to zero: $x^* = y/2$.
If $y \in \mathbb{R}^2_{++}$, then $x^* = y/2 \in \mathbb{R}^2_{++}$ and $f^*(y) = \|y\|^2/4$.
$$\boxed{f^*(y) = \frac{\|y\|_2^2}{4}}$$

(b) $f(x) = |x|^3/3$, $x \in \mathbb{R}$

中文：
$f^*(y) = \sup_x [yx - |x|^3/3]$
当 $x \geq 0$：$g(x) = yx - x^3/3$，$g'(x) = y - x^2 = 0 \Rightarrow x = \sqrt{y}$（需 $y \geq 0$）
代入：$g(\sqrt{y}) = y\sqrt{y} - y\sqrt{y}/3 = \frac{2}{3}y^{3/2}$
当 $x < 0$：$g(x) = yx + x^3/3$（因为 $|x|^3 = -x^3$ 当 $x<0$... 不对，$|x|^3 = |x|^3 = (-x)^3 = -x^3$... 实际上 $|x|^3 = |x|^3$，当 $x<0$ 时 $|x| = -x$，所以 $|x|^3 = (-x)^3 = -x^3$）
所以 $f(x) = |x|^3/3$，当 $x < 0$ 时 $f(x) = (-x)^3/3 = -x^3/3$。
$g(x) = yx - (-x^3/3) = yx + x^3/3$，$g'(x) = y + x^2 = 0 \Rightarrow x^2 = -y$（需 $y \leq 0$），$x = -\sqrt{-y}$
代入：$g(-\sqrt{-y}) = y(-\sqrt{-y}) + (-\sqrt{-y})^3/3 = -y\sqrt{-y} - (-y)^{3/2}/3 = -y\sqrt{-y} + y\sqrt{-y}/3$...
让我用 $|y|$ 重写：当 $y \leq 0$，令 $y = -|y|$，$x = -\sqrt{|y|}$：
$g = -|y| \cdot (-\sqrt{|y|}) + (-\sqrt{|y|})^3/3 = |y|^{3/2} - |y|^{3/2}/3 = \frac{2}{3}|y|^{3/2}$

综合：$$\boxed{f^*(y) = \frac{2}{3}|y|^{3/2}}$$ English：
For $y \geq 0$: optimal $x = \sqrt{y}$, giving $f^*(y) = \frac{2}{3}y^{3/2}$.
For $y < 0$: optimal $x = -\sqrt{|y|}$, giving $f^*(y) = \frac{2}{3}|y|^{3/2}$.
$$\boxed{f^*(y) = \frac{2}{3}|y|^{3/2}}$$

Problem 3: 对偶问题 / Duality

题目：maximize $f(x) = x^Tx$, subject to $Ax = b$

(a) 拉格朗日函数 / Lagrangian

中文：将最大化问题转为最小化：minimize $-x^Tx$ s.t. $Ax = b$
$$\boxed{L(x, \nu) = -x^Tx + \nu^T(Ax - b)}$$ English：Convert to minimization: minimize $-x^Tx$ s.t. $Ax = b$.
$L(x, \nu) = -x^Tx + \nu^T(Ax - b)$

(b) 对偶问题 / Dual problem

中文：
对偶函数：$g(\nu) = \inf_x [-x^Tx + \nu^TAx - \nu^Tb]$
对 $x$ 求导令为0：$-2x + A^T\nu = 0 \Rightarrow x = \frac{1}{2}A^T\nu$
代入： $$g(\nu) = -\frac{1}{4}\nu^TAA^T\nu + \frac{1}{2}\nu^TAA^T\nu - \nu^Tb = \frac{1}{4}\nu^TAA^T\nu - \nu^Tb$$ 对偶问题：$$\boxed{\text{maximize} \quad \frac{1}{4}\nu^TAA^T\nu - \nu^Tb}$$ English：
Dual function: $g(\nu) = \inf_x[-x^Tx + \nu^TAx - \nu^Tb]$.
FOC: $-2x + A^T\nu = 0 \Rightarrow x = A^T\nu/2$.
Substituting: $g(\nu) = \frac{1}{4}\nu^TAA^T\nu - \nu^Tb$.
Dual problem: maximize $\frac{1}{4}\nu^TAA^T\nu - \nu^Tb$.

(c) KKT条件 / KKT conditions

中文/English：
  1. 原始可行性 / Primal feasibility: $Ax^* = b$
  2. 梯度条件 / Stationarity: $-2x^* + A^T\nu^* = 0$，即 $x^* = \frac{1}{2}A^T\nu^*$
（无不等式约束，所以没有对偶可行性和互补松弛条件）
(No inequality constraints, so no dual feasibility or complementary slackness.)

(d) 强对偶性 / Strong duality

中文：
注意：原问题是最大化凸函数（$x^Tx$ 是凸的），这不是凸优化问题！它是非凸的。
如果 $A$ 的零空间非平凡（即存在 $z \neq 0$ 使得 $Az = 0$），则对任何可行点 $x_0$（$Ax_0 = b$），$x_0 + tz$ 也可行，且 $\|x_0 + tz\|^2 \to \infty$。所以 $p^* = +\infty$（最优值无界）。
如果 $A$ 列满秩（零空间只有 $\{0\}$），则可行集只有一个点 $x = A^{\dagger}b$（如果 $m \geq n$），$p^* = \|A^{\dagger}b\|^2$。

一般情况下，如果可行集无界，则 $p^* = +\infty$。

English：
The primal maximizes a convex function — this is NOT a convex optimization problem!
If $A$ has a nontrivial null space (i.e., $\exists z \neq 0$ with $Az = 0$), then for any feasible $x_0$, $x_0 + tz$ is also feasible and $\|x_0+tz\|^2 \to \infty$ as $t \to \infty$. So $p^* = +\infty$ (unbounded).
If $A$ has full column rank and $m \geq n$, the feasible set is a single point and $p^*$ is finite.
Strong duality may or may not hold depending on the structure of $A$.

Problem 4: 梯度下降与线搜索 / Gradient Descent & Line Search

(a) 梯度下降算法 / Gradient Descent Algorithm

中文：
输入：初始点 $x^{(0)} \in \text{dom}(f)$，容差 $\epsilon > 0$
重复：
1. 计算梯度 $\nabla f(x)$
2. 如果 $\|\nabla f(x)\|_2 \leq \epsilon$，停止
3. 设下降方向 $\Delta x = -\nabla f(x)$
4. 通过线搜索选择步长 $t > 0$
5. 更新 $x := x + t\Delta x$

English：
Input: starting point $x^{(0)} \in \text{dom}(f)$, tolerance $\epsilon > 0$
Repeat:
1. Compute gradient $\nabla f(x)$
2. If $\|\nabla f(x)\|_2 \leq \epsilon$, stop
3. Set descent direction $\Delta x = -\nabla f(x)$
4. Choose step size $t > 0$ via line search
5. Update $x := x + t\Delta x$

(b) 精确线搜索与回溯线搜索 / Exact & Backtracking Line Search

精确线搜索 / Exact：$t = \arg\min_{t>0} f(x + t\Delta x)$
沿搜索方向找使目标函数最小的步长。/ Find the step size minimizing $f$ along the search direction.

回溯线搜索 / Backtracking：
参数 $\alpha \in (0, 0.5)$, $\beta \in (0, 1)$。从 $t = 1$ 开始：
While $f(x + t\Delta x) > f(x) + \alpha t \nabla f(x)^T\Delta x$: 令 $t := \beta t$
Parameters $\alpha \in (0,0.5)$, $\beta \in (0,1)$. Start with $t=1$. While Armijo condition is violated, shrink: $t := \beta t$.

(c) 最速下降方向 / Steepest descent directions

归一化最速下降方向 / Normalized： $$\Delta x_{\text{nsd}} = \arg\min_{\|v\| = 1} \nabla f(x)^T v$$ 非归一化最速下降方向 / Unnormalized： $$\Delta x_{\text{sd}} = \arg\min_{\|v\| \leq 1} \nabla f(x)^T v$$ 对于欧几里得范数 / For Euclidean norm: $\Delta x_{\text{nsd}} = -\nabla f(x)/\|\nabla f(x)\|_2$, $\Delta x_{\text{sd}} = -\nabla f(x)$

(d) 证明 $g(x) = -\ln(x)$ 自协调 / Prove $g(x) = -\ln(x)$ is self-concordant

中文：对 $g(x_1, x_2) = -\ln x_1 - \ln x_2$，$x \in \mathbb{R}^2_{++}$：
由于函数可分离，只需证明一维情况 $h(t) = -\ln t$ 是自协调的。
$h'(t) = -1/t$, $h''(t) = 1/t^2$, $h'''(t) = -2/t^3$
验证：$|h'''(t)| = 2/t^3$, $2(h''(t))^{3/2} = 2(1/t^2)^{3/2} = 2/t^3$
$|h'''(t)| = 2(h''(t))^{3/2}$ ✓（等号成立）
自协调函数的和仍是自协调的，所以 $g(x_1,x_2) = -\ln x_1 - \ln x_2$ 自协调。✓

English：
Since $g$ is separable, it suffices to show $h(t) = -\ln t$ is self-concordant.
$h''(t) = 1/t^2$, $h'''(t) = -2/t^3$.
Check: $|h'''(t)| = 2/t^3 = 2(1/t^2)^{3/2} = 2(h''(t))^{3/2}$ ✓
Sum of self-concordant functions is self-concordant, so $g$ is self-concordant. ✓

Problem 5: 强凸函数不等式 / Strongly convex inequality

题目：$f$ 强凸，$\nabla^2 f(x) \succeq mI$ 在 $S = \{x \mid f(x) \leq f(x^{(0)})\}$ 上。证明 $f(x) - p^* \leq \frac{1}{2m}\|\nabla f(x)\|_2^2$。

中文证明：
由强凸性，对任意 $y$： $$f(y) \geq f(x) + \nabla f(x)^T(y-x) + \frac{m}{2}\|y-x\|^2$$ 右边是关于 $y$ 的二次函数，对 $y$ 求最小值：
令 $\nabla_y [f(x) + \nabla f(x)^T(y-x) + \frac{m}{2}\|y-x\|^2] = \nabla f(x) + m(y-x) = 0$
得 $y^* = x - \frac{1}{m}\nabla f(x)$
代入右边的最小值： $$f(y^*) \geq f(x) + \nabla f(x)^T(-\frac{1}{m}\nabla f(x)) + \frac{m}{2}\|\frac{1}{m}\nabla f(x)\|^2 = f(x) - \frac{1}{2m}\|\nabla f(x)\|^2$$ 由于 $p^* = \inf_y f(y) \leq f(y)$ 对所有 $y$，但上面的不等式方向是 $f(y) \geq$ 右边。
正确推导：由于对所有 $y$ 都有 $f(y) \geq f(x) - \frac{1}{2m}\|\nabla f(x)\|^2$... 不对。

正确推导：
对最优点 $x^*$（$f(x^*) = p^*$）应用强凸性： $$p^* = f(x^*) \geq f(x) + \nabla f(x)^T(x^* - x) + \frac{m}{2}\|x^* - x\|^2$$ 这给出 $p^*$ 的下界，但我们需要上界。换个思路：

由强凸性的等价形式：$f(y) \geq f(x) + \nabla f(x)^T(y-x) + \frac{m}{2}\|y-x\|^2$ 对所有 $y$。
取右边关于 $y$ 的最小值（这是 $f$ 的一个下界的最小值）：
$$p^* = \min_y f(y) \geq \min_y \left[f(x) + \nabla f(x)^T(y-x) + \frac{m}{2}\|y-x\|^2\right]$$ 右边最小值在 $y = x - \frac{1}{m}\nabla f(x)$ 取到，值为 $f(x) - \frac{1}{2m}\|\nabla f(x)\|^2$。
所以 $p^* \geq f(x) - \frac{1}{2m}\|\nabla f(x)\|^2$，即： $$\boxed{f(x) - p^* \leq \frac{1}{2m}\|\nabla f(x)\|_2^2}$$ ✓

English Proof：
By strong convexity: $f(y) \geq f(x) + \nabla f(x)^T(y-x) + \frac{m}{2}\|y-x\|^2$ for all $y$.
Taking the minimum over $y$ on both sides:
$p^* = \min_y f(y) \geq \min_y [f(x) + \nabla f(x)^T(y-x) + \frac{m}{2}\|y-x\|^2]$
The RHS minimizer is $y^* = x - \frac{1}{m}\nabla f(x)$, giving minimum value $f(x) - \frac{1}{2m}\|\nabla f(x)\|^2$.
Therefore: $p^* \geq f(x) - \frac{1}{2m}\|\nabla f(x)\|^2$, i.e., $f(x) - p^* \leq \frac{1}{2m}\|\nabla f(x)\|^2$. ✓

Problem 6: 牛顿法 / Newton's Method

(a) 牛顿下降方向 / Newton descent direction

中文：牛顿下降方向定义为： $$\boxed{\Delta x_{\text{nt}} = -[\nabla^2 f(x)]^{-1}\nabla f(x)}$$ 它是在当前点 $x$ 处，对 $f$ 做二阶Taylor展开后，最小化该二次近似得到的方向。

English：The Newton descent direction is $\Delta x_{\text{nt}} = -[\nabla^2 f(x)]^{-1}\nabla f(x)$. It is the direction obtained by minimizing the second-order Taylor approximation of $f$ at $x$.

(b) 牛顿减量 / Newton decrement $\lambda(x)$

$$\boxed{\lambda(x) = \left(\nabla f(x)^T [\nabla^2 f(x)]^{-1} \nabla f(x)\right)^{1/2} = \left(\Delta x_{\text{nt}}^T \nabla^2 f(x) \Delta x_{\text{nt}}\right)^{1/2}}$$

中文：牛顿减量是一个仿射不变的量，衡量当前点到最优点的"距离"。当 $\lambda(x) \ll 1$ 时，牛顿法进入二次收敛阶段。

English：The Newton decrement is an affine-invariant measure of proximity to the optimum. When $\lambda(x) \ll 1$, Newton's method enters the quadratically convergent phase.

(c) 二阶近似的最小值点和最小值 / Minimizer and minimum of quadratic approximation

题目：$g(y) = f(x) + \nabla f(x)^Ty + \frac{1}{2}y^T\nabla^2 f(x)y$ 是 $f(x+y)$ 在 $x$ 点的二阶近似。

最小值点 / Minimizer：
$\nabla_y g = \nabla f(x) + \nabla^2 f(x) y = 0$
$$\boxed{y^* = -[\nabla^2 f(x)]^{-1}\nabla f(x) = \Delta x_{\text{nt}}}$$ The minimizer of the quadratic approximation is exactly the Newton step.

最小值 / Minimum value：
$$g(y^*) = f(x) + \nabla f(x)^T\Delta x_{\text{nt}} + \frac{1}{2}\Delta x_{\text{nt}}^T\nabla^2 f(x)\Delta x_{\text{nt}}$$ $$= f(x) - \nabla f(x)^T[\nabla^2 f]^{-1}\nabla f(x) + \frac{1}{2}\nabla f(x)^T[\nabla^2 f]^{-1}\nabla f(x)$$ $$\boxed{= f(x) - \frac{1}{2}\lambda^2(x)}$$

(d) 证明 $f(x) - \inf_y g(y) = \frac{1}{2}\lambda^2(x)$

中文：由(c)的结果：$\inf_y g(y) = g(y^*) = f(x) - \frac{1}{2}\lambda^2(x)$
因此： $$\boxed{f(x) - \inf_y g(y) = f(x) - [f(x) - \frac{1}{2}\lambda^2(x)] = \frac{1}{2}\lambda^2(x)}$$ ✓

English：From (c): $\inf_y g(y) = f(x) - \frac{1}{2}\lambda^2(x)$.
Therefore: $f(x) - \inf_y g(y) = \frac{1}{2}\lambda^2(x)$. ✓

直觉 / Intuition：$\frac{1}{2}\lambda^2(x)$ 是当前函数值与二阶近似最优值之间的差距，反映了"如果世界是二次的，牛顿一步能下降多少"。
$\frac{1}{2}\lambda^2(x)$ represents how much Newton's method would decrease the function if the quadratic approximation were exact.
